# 🔐 TGM Bridge (Client App)

Private family messaging test app.

This app allows:
- Secure login (Firebase Authentication)
- Send message to Admin
- Receive replies
- Encode / Decode messages
- PIN protection

---

## 🧩 Requirements

- Python 3.9 or higher
- Internet connection
